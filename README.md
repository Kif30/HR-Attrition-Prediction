# HR Attrition Prediction (Machine Learning Project)

## 📌 Project Overview

This project predicts whether an employee will leave the company using machine learning.

## 🎯 Objective

To analyze employee data and build classification models for attrition prediction.

## 🛠 Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## 📊 Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* KNN

## ⚙️ Techniques Applied

* Data Preprocessing
* Feature Encoding
* Feature Scaling
* SMOTE
* Model Evaluation
* Hyperparameter Tuning

## 📈 Result

Random Forest performed best with highest F1-score.

## 💡 Key Insights

* Overtime increases attrition
* Low job satisfaction increases attrition
* Poor work-life balance affects retention

## 🚀 Conclusion

Machine learning helps predict employee attrition and supports better business decisions.
